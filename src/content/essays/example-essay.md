This is a placeholder essay. You can create new essays by adding `.md` files to the `content/essays/` folder.

Each essay needs frontmatter at the top with:
- **title**: The essay heading
- **date**: Publication date (YYYY-MM-DD format)
- **description**: A one-line summary

The essay body can include markdown formatting: *italics*, **bold**, links, code blocks, lists, and more.

To create a new essay, just add a new `.md` file with the same frontmatter structure. Save and push to GitHub, and it will appear automatically on your writing page.
